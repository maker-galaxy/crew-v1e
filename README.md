# crew-v1e

Content and configuration for the **V1 Engineering crew** in Maker Galaxy Explorer.

This repository defines the offline content pack used by Explorer tablets at events and exhibitions.

## What lives here

- Community messaging and page content
- Images, previews, and media used by Explorer
- Topic and video mappings
- Crew configuration and content pack manifest

## What does NOT live here

Explorer core code, gallery engine, scripts, and shared platform components live in the Maker Galaxy core repositories.

## Editing

Community owners can edit content directly using the GitHub web editor.

Most edits happen in:

```

explorer/content/
explorer/mappings/
explorer/assets/

```

Changes are packaged into an Explorer **content pack** and synced to tablets.

## Manifest

`crew-manifest.json` defines the entry point used by the Maker Galaxy content pipeline.
